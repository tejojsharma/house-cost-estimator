# Modern House Cost Estimator

A clean, user-friendly desktop application built with Python and Tkinter that helps users quickly estimate the total construction cost of a residential building based on area, number of floors, and desired house style.

## Overview

This tool is designed for homeowners, architects, real estate enthusiasts, or anyone looking for a fast and approximate cost calculation before starting a construction project. It provides instant feedback with a modern and intuitive graphical interface.

## Key Features

- Input total built-up area in square feet
- Specify the number of floors (ground + additional floors)
- Choose from three predefined construction categories:
  - Simple House
  - Duplex House
  - Luxury Villa
- Real-time cost estimation with detailed breakdown
- Input validation with clear error messages
- Responsive and professionally styled user interface
- Lightweight — no external dependencies beyond standard Python

## Cost Rates Used (per square foot)

| House Type   | Cost per Sqft | Description                     |
|--------------|---------------|---------------------------------|
| Simple       | ₹1,500        | Basic finishes, standard materials |
| Duplex       | ₹2,050        | Moderate upgrades, better aesthetics |
| Luxury       | ₹2,600        | Premium materials and finishes     |

> Note: These rates are approximate averages for estimation purposes only and may vary by region, contractor, and market conditions.


## Requirements

- Python 3.6 or newer
- Tkinter (comes pre-installed with standard Python)
